1 - Diferenciar as camadas 2 e 3 do modelo OSI, e indicar os protocolos utilizados para endereçamento nestas camadas.

A Camada 2 (Enlace) gerencia a comunicação dentro de uma mesma rede local, usando o endereço físico MAC para entregar 
dados entre dispositivos diretamente conectados. Já a Camada 3 (Rede) é responsável pela comunicação entre redes 
diferentes (inter-rede), utilizando o endereço lógico IP para rotear pacotes de dados pela internet e encontrar o 
melhor caminho até o destino.


2 - Qual a diferença entre adotar uma solução proprietária como o sistema operacional Windows quando comparado a adoção 
de uma solução OpenSource como o sistema operacional Ubuntu? Quais seriam os pontos negativos e positivos de cada 
abordagem?

A principal diferença é o modelo de licença e acesso ao código. Windows é um software proprietário de código fechado, 
enquanto Ubuntu é um sistema de código aberto. Positivamente, Windows oferece amplo suporte e compatibilidade de 
software, mas negativamente, possui custos de licença e menor flexibilidade. Ubuntu se destaca por ser gratuito, seguro 
e customizável, mas pode ter uma curva de aprendizado e menor compatibilidade com softwares específicos.


3 - O que seria um projeto Open Source? Como empresas podem adotar tais tecnologias e o que isso acarreta?

Um projeto Open Source tem seu código-fonte publicamente disponível, permitindo que qualquer pessoa o use, estude e 
modifique, regido por uma licença específica. Empresas adotam essa tecnologia utilizando-a em sua infraestrutura 
(ex: servidores Linux), contribuindo com projetos ou abrindo o código de suas próprias ferramentas. Isso acarreta em 
grande redução de custos e inovação acelerada, mas exige responsabilidade no gerenciamento de segurança e conformidade 
com as licenças.
